## Linha do tempo
- 2026-05-11 - HOTFIX produção: app em https://mw-personal-trainer.web.app ficou em branco apos os deploys de Sprint 2/3. Causa: builds rodaram sem `VITE_BASE_PATH=/` setado, vite.config.js usava default `/pt/`, e o `index.html` source tinha paths hardcoded `/pt/manifest.json`, `/pt/sw.js`, `/pt/icons/icon.svg`. Em Firebase Hosting (raiz), o navegador pedia `/pt/assets/index-*.js` e o catch-all rewrite `**` -> `/index.html` devolvia HTML (1172 bytes) como se fosse JS — `Uncaught SyntaxError: Unexpected token '<'` e tela em branco. Correcao em 3 partes: `index.html` usa `%BASE_URL%manifest.json` / `%BASE_URL%sw.js` / `%BASE_URL%icons/icon.svg` / `scope:"%BASE_URL%"` (Vite substitui no build); `vite.config.js` passou a usar `loadEnv` para ler `VITE_BASE_PATH` de `.env.production`; criado `.env.production` versionado com `VITE_BASE_PATH=/`. Dev local continua usando default `/pt/` (Caddy proxy preservado). Deploy `firebase deploy --only hosting` em 2026-05-11. Smoke: `/assets/index-*.js` retorna `text/javascript` 388 KB; `/manifest.json` retorna `application/json`.
- 2026-05-09 - Sprint 3 do plano de melhorias (Custo & Pipeline): (Item #3) novo modulo `firebase/tokenBudget.js` com `checkBudget`/`debit`/`getStatus`/`resetBudget` operando contadores diario+mensal em `users/{uid}/tokenBudgets/daily_YYYYMMDD` e `monthly_YYYYMM` com FieldValue.increment atomico. Defaults `TOKEN_BUDGET_DAILY=500000`, `TOKEN_BUDGET_MONTHLY=5000000`, configuraveis por env. Pre-check em `routes/firebaseClaude.js` antes de `enqueueClaudeTask` retorna 429 + `{code:"TOKEN_BUDGET_EXCEEDED", scope, resetAt, usage}` quando estoura. Debito em `firebase/worker.js` apos `response.ok` na mesma transacao do `aiLog`. Endpoint self-service `GET /api/token-budget` e admin `GET/POST /api/admin/token-budget/:uid[/reset]`. TTL de 48h (daily) e 35d (monthly) via index `tokenBudgets.expiresAt`. (Item #4) novo workflow GitHub Actions `.github/workflows/deploy.yml` com 4 jobs (lint, test, build, deploy) — deploy so em `main` e atras de `environment: production` (required reviewer).
